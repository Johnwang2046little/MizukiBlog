## 1. 基本样式

### 1.1 标题

使用`#` 可以创建出标题。

示例：

```markdown
# 一级标题
## 二级标题
### 三级标题
#### 四级标题
##### 五级标题
###### 六级标题
没了
```

因为会影响到目录，所以这里不做演示。上面其实就有标题了。

### 1.2 段落

行尾加两个空格可以换行。（许多编辑器带有自动换行）

按两下回车可以分段。

示例：

```markdown
我是第一段  
换行了


我是第二段
```



分割线下面是演示：

---

我是第一段  
换行了

我是第二段

### 1.4 引用

在文本前加上`> `进行引用。

示例：

```markdown
> 鲁迅曾经说过：“糯米粽的博客超棒的！”
```

分割线下面是演示：

---

> 鲁迅曾经说过：“糯米粽的博客超棒的！”

## 2. 进阶样式

### 2.1 无序列表

使用`-` `*` `+` 构建无序列表。

示例：

```markdown
下面是一个列表：
- Jiamin
- Janet
- Xiaoling
- 李华
```

演示：

---

下面是一个列表：

- Jiamin
- Janet
- Xiaoling
- 李华

### 2.2 有序列表

使用`1.` 等开头的文本构建有序列表。

示例：

```md
1. Jiamin
2. Janet
8. 飞起来！
1000. 列表是不会乱的！
```

演示：

---

1. Jiamin
2. Janet
3. 飞起来！
4. 列表是不会乱的！

都看到了吧，只要有了1，后面都不会乱掉的了。

### 2.3 行内代码

使用反引号`包裹代码。

示例：

```md
下面介绍打印函数`print()`。
```

演示：

---

下面介绍打印函数`print()`。

### 2.5 链接

方括号内是显示的文字，括号内是链接。

示例：

```md
[点进来看看](https://www.bilibili.com/video/BV1GJ411x7h7/)
```

演示：

---

[点进来看看](https://www.bilibili.com/video/BV1GJ411x7h7/)

### 2.6 图片

示例：

```md
![替代文本（加载失败时显示）](https://p1.ssl.qhmsg.com/t0143b426aee3bb03cb.jpg)
```

演示：

---

![替代文本（加载失败时显示）](https://p1.ssl.qhmsg.com/t0143b426aee3bb03cb.jpg)

不是你好占地方啊

### 2.7 表格

表格有一丢丢复杂，可以看看格式：

示例：

```md
| 左对齐 | 居中对齐 | 右对齐 |
| :----- | :------: | -----: |
| 内容1  | 内容2    | 内容3  |
| 长文本内容 | 居中示例 | 数值100 |
```

演示：

---

| 左对齐   | 居中对齐 | 右对齐   |
|:----- |:----:| -----:|
| 内容1   | 内容2  | 内容3   |
| 长文本内容 | 居中示例 | 数值100 |

网站上展示不了对齐方式，但是Markdown有这个语法。
